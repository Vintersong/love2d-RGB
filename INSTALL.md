# CHROMATIC — Installation & Launch Guide

A bullet-heaven built in LÖVE 11.5. Single-player. ~10–20 min per run.

## Windows (recommended for judging)
1. Download and unzip `chromatic-windows.zip`.
2. Double-click **`chromatic.exe`**. No installation required.
   (Keep `chromatic.exe` together with the bundled LÖVE `.dll` files and `license.txt`.)
- Requires 64-bit Windows 10/11. No internet connection needed.

## Browser (online build)
- Open the provided web link (or `dist/web/index.html`) in a desktop Chrome/Edge/Firefox.
- Allow audio when prompted; click the canvas, then press **SPACE** to start.

## From source (any OS)
1. Install **LÖVE 11.5** from https://love2d.org.
2. From the repository root, run: `love .`

## Controls
| Input | Action |
|-------|--------|
| **WASD** | Move |
| **Mouse** | Aim (auto-fire toward cursor) |
| **Space** | Dash (also: start the run from the splash screen) |
| **E** | Blink — teleport to the mouse position |
| **P / Esc** | Pause / resume |

## How to play
Survive waves, collect XP orbs to level up, and commit to two primary colors to shape your
build. Bosses appear periodically; the **final boss is a four-phase ring boss** — deplete its
health bar (each quarter is a phase) and destroy the exposed core in the last phase to win.
A run ends in victory (boss defeated) or game over (you die).
