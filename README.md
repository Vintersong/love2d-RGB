LÖVE2D RGB Bullet Hell Game

A small bullet hell game built with LÖVE2D featuring a color-based progression system (RGB → YMC), stackable combat mechanics, and music-reactive spawning.

## Play in Browser

The game is automatically built and deployed to GitHub Pages on every push to `main`. Once the repository has GitHub Pages enabled, you can play it at:

`https://<username>.github.io/love2d-RGB/`

No installation required — just open the link in a modern desktop browser.

## Play Locally

Install [LÖVE2D 11.x](https://love2d.org/) and run:

```bash
love .
```

Features

8 unique artifacts with different behaviors

Rapid prototyping workflow using AI-assisted coding

Designed for experimentation with gameplay systems

Controls

Arrow keys / WASD: Move

Space: Dash

This project is primarily for demonstration purposes and resume portfolio. Debug overlays are currently active.
